## What is This?

## The 6 AI Analysts

жҜҸдҪҚеҲҶжһҗеё«йғҪжңүзҚЁз«Ӣзҡ„ SKILL.md е®ҡзҫ©е…¶и§’иүІгҖҒеҲҶжһҗжЎҶжһ¶е’ҢијёеҮәж јејҸпјҡ

### рҹ’° иІЎеӢҷеҲҶжһҗеё« Financial Analyst

> еҹәжң¬йқўеҲҶжһҗпјҡзҚІеҲ©иғҪеҠӣгҖҒдј°еҖјж°ҙжә–гҖҒиІЎеӢҷзөҗж§ӢгҖҒиӮЎеҲ©ж”ҝзӯ–

- PE / PB / PEG / Forward PE дј°еҖјжҜ”ијғ
- ROEгҖҒROAгҖҒжҜӣеҲ©зҺҮгҖҒж·ЁеҲ©зҺҮеҲҶжһҗ
- иІ еӮөжҜ”гҖҒжөҒеӢ•жҜ”зҺҮгҖҒйҖҹеӢ•жҜ”зҺҮеҒҘеә·еәҰ
- иӮЎеҲ©ж®–еҲ©зҺҮгҖҒй…ҚзҷјзҺҮж°ёзәҢжҖ§и©•дј°

### рҹ“Ҳ жҠҖиЎ“еҲҶжһҗеё« Technical Analyst

> еғ№йҮҸеҲҶжһҗпјҡи¶ЁеӢўеҲӨж–·гҖҒж”Ҝж’җеЈ“еҠӣгҖҒеӢ•иғҪжҢҮжЁҷ

- еқҮз·ҡзі»зөұпјҲMA5/10/20/50/200пјүжҺ’еҲ—иҲҮи¶ЁеӢў
- RSI и¶…иІ·и¶…иіЈгҖҒMACD дәӨеҸүгҖҒKD й»ғйҮ‘/жӯ»дәЎдәӨеҸү
- еёғжһ—йҖҡйҒ“еЈ“зё®иҲҮзӘҒз ҙ
- й—ңйҚөж”Ҝж’җеЈ“еҠӣдҪҚиҫЁиӯҳ

### рҹ”ў йҮҸеҢ–еҲҶжһҗеё« Quantitative Analyst

> йўЁйҡӘе ұй…¬еҲҶжһҗпјҡжіўеӢ•еәҰгҖҒеӨҸжҷ®жҜ”зҺҮгҖҒжңҖеӨ§еӣһж’Ө

- е№ҙеҢ–е ұй…¬зҺҮ vs еёӮе ҙеҹәжә–
- жіўеӢ•зҺҮи©•зҙҡиҲҮжңҖеӨ§еӣһж’ӨеҲҶжһҗ
- Sharpe / Sortino Ratio йўЁйҡӘиӘҝж•ҙеҫҢе ұй…¬
- жғ…еўғеҲҶжһҗпјҲжЁӮи§Җ / еҹәжң¬ / жӮІи§Җпјү

### рҹҸӯ з”ўжҘӯзёҪз¶“еҲҶжһҗеё« Industry & Macro Analyst

> з”ўжҘӯе®ҡдҪҚпјҡз«¶зҲӯж јеұҖгҖҒжҷҜж°ЈеҫӘз’°гҖҒзёҪз¶“з’°еўғ

- з”ўжҘӯең°дҪҚиҲҮеёӮдҪ”зҺҮеҲҶжһҗ
- з«¶зҲӯиӯ·еҹҺжІіи©•дј°
- жҷҜж°ЈеҫӘз’°йҡҺж®өеҲӨж–·
- еҲ©зҺҮгҖҒеҢҜзҺҮгҖҒең°з·Јж”ҝжІ»еҪұйҹҝи©•дј°

### рҹ“° ж–°иҒһжғ…з·’еҲҶжһҗеё« News Sentiment Analyst

> еёӮе ҙжғ…з·’пјҡж–°иҒһеҲҶйЎһгҖҒйҮҚеӨ§дәӢд»¶гҖҒж•ҳдәӢж–№еҗ‘

- йҖҗзҜҮж–°иҒһжғ…з·’еҲҶйЎһпјҲжӯЈйқў/дёӯжҖ§/иІ йқўпјү
- жғ…з·’жҢҮж•ёпјҲ-10 еҲ° +10пјүеҠ ж¬ҠиЁҲз®—
- йҮҚеӨ§дәӢд»¶еҒөжё¬пјҲиІЎе ұгҖҒдҪөиіјгҖҒзӣЈз®Ўпјү

## Installation

### Prerequisites

- **Python 3.9+**
- **Claude Code CLI** (with Claude Pro / Team / Enterprise subscription)

### Step 1: Clone & Install

```bash
git clone https://github.com/liiandy/stock-analysis-system.git
cd stock-analysis-system
chmod +x install.sh
./install.sh
```

The install script will:
1. Install Python dependencies (`yfinance`, `pandas`, `numpy`)
2. Copy all 11 Skills + `shared/` to `~/.claude/skills/`
3. Replace path placeholders with your machine's absolute paths
4. Create output directory at `~/fubon-stock-reports/`
5. Write config to `~/.claude/stock-analysis.conf`
6. **Ask once** whether to auto-configure Claude Code permissions (WebSearch, WebFetch, Bash)

If you answer **Y** to the permission prompt, the plugin will run fully automatically without any manual approval popups. All permissions are:
- **Read-only** (WebFetch only reads web pages, never uploads)
- **Scoped** to plugin scripts and financial news domains only
- **Removable** вҖ” cleanly removed on uninstall

### Options

```bash
# Custom output directory
./install.sh --output-dir ~/Desktop/stock-reports

# Skip permission prompt (auto-approve, for CI/automation)
./install.sh --auto-permissions
```

### Step 2: Verify вҖ” Open Claude Code in **ANY** Directory

```bash
cd ~
claude   # or open Claude Code in any folder
```

Type:

```
еҲҶжһҗеҸ°з©Қйӣ»
```

If everything is set up correctly, the system will begin the full analysis pipeline. Reports will be saved to `~/fubon-stock-reports/tsmc/`.

### Uninstall

```bash
./uninstall.sh                    # Keep reports & remove permissions
./uninstall.sh --remove-reports   # Remove everything including reports
./uninstall.sh --keep-permissions # Keep permissions in settings.json
```

Uninstall automatically removes all plugin permissions from `~/.claude/settings.json` (identified by `# __stock-analysis-plugin__` marker) without affecting your other settings.

## Pipeline Deep Dive

е®Ңж•ҙзҡ„ 7 жӯҘй©ҹеҲҶжһҗжөҒзЁӢпјҡ

### Step 1: и§ЈжһҗдҪҝз”ЁиҖ…ж„Ҹең– & жЁЎејҸеҲҶйЎһ

Orchestrator еҫһиҮӘз„¶иӘһиЁҖдёӯжҸҗеҸ– tickerпјҢж”ҜжҸҙе…¬еҸёеҗҚзЁұпјҲйҙ»жө·пјүеҲ° tickerпјҲ2317.TWпјүзҡ„иҮӘеӢ•жҳ е°„гҖӮиӢҘе…¬еҸёеҗҚдёҚзўәе®ҡпјҢиҮӘеӢ• WebSearch жҹҘи©ўжӯЈзўә tickerгҖӮ

жҺҘи‘—еҲҶйЎһзӮәдёүзЁ®жЁЎејҸд№ӢдёҖпјҡ
- **quick_answer**пјҡе–®дёҖж•ёж“ҡжҹҘи©ў вҶ’ `quick_quote.py` зӣҙжҺҘеӣһзӯ”пјҢи·ійҒҺеҫҢзәҢжӯҘй©ҹ
- **selective**пјҡзү№е®ҡз¶ӯеәҰеҲҶжһҗ вҶ’ еҸӘе•ҹеӢ•зӣёй—ң Agent
- **full_analysis**пјҡе®Ңж•ҙ 6 Agent еҲҶжһҗ

### Step 2: иіҮж–ҷжҠ“еҸ– & й©—иӯүпјҲеҗҲдҪөи…іжң¬пјү

```
fetch_and_validate.py вҶ’ validated_data.json (+ optional raw_data.json)
```

- `fetch_data.py` е…§йғЁдҪҝз”Ё **ThreadPoolExecutor** дёҰиЎҢе‘јеҸ« 7 еҖӢ APIпјҲ~3-4 з§’пјҢеҺҹжң¬ ~8-12 з§’пјү
- `validate_data.py` еңЁеҗҢдёҖйҖІзЁӢдёӯзӣҙжҺҘиҷ•зҗҶпјҢзңҒеҺ»дёҖж¬Ў Python еҶ·е•ҹеӢ• + JSON еәҸеҲ—еҢ–
- иҮӘеӢ•еҒөжё¬еёӮе ҙдёҰеҘ—з”Ёе°ҚжҮүй–ҫеҖјпјҲеҰӮеҸ°иӮЎжјІи·ҢеҒң 11%пјү

### Step 3: дёүеұӨиіҮж–ҷе“ҒиіӘй–ҳй–Җ

| Tier | Confidence | Action |
|------|-----------|--------|
| `hard_stop` | < 30% | еҒңжӯўеҲҶжһҗпјҢйҖҡзҹҘдҪҝз”ЁиҖ… |
| `warning` | 30-49% | иӯҰе‘ҠдҪҶз№јзәҢпјҢйҷҚдҪҺдҝЎеҝғй җжңҹ |
| `passed` | вүҘ 50% | жӯЈеёёйҖІиЎҢ |

### Step 3.5: Agent еҝ«еҸ–жӘўжҹҘ & Quant й җе•ҹеӢ•

- жӘўжҹҘеҗҢж—Ҙе·Іе®ҢжҲҗзҡ„ Agent еҲҶжһҗпјҢзӣҙжҺҘеҫ©з”ЁпјҲselective вҶ’ full е ҙжҷҜзү№еҲҘжңүж•Ҳпјү
- `analyze_quant.py` еңЁиғҢжҷҜе•ҹеӢ•пјҢиҲҮ Agent дёҰиЎҢйҒӢз®—

### Step 4: AI еҲҶжһҗеё«дёҰиЎҢеҲҶжһҗ

Claude Agent еҗҢжҷӮе•ҹеӢ•пјҲжүЈйҷӨеҝ«еҸ–е‘Ҫдёӯзҡ„пјүпјҢеҗ„иҮӘжҺҘж”¶зӣёй—ңж•ёж“ҡдёҰйҖІиЎҢе°ҲжҘӯеҲҶжһҗгҖӮжҜҸдҪҚеҲҶжһҗеё«ијёеҮәзҚЁз«Ӣзҡ„ JSON е ұе‘ҠпјҢеҢ…еҗ«пјҡ

- **Score** (0-10)пјҡи©Із¶ӯеәҰи©•еҲҶ
- **Confidence**пјҡдҝЎеҝғж°ҙжә–
- **Summary**пјҡз№Ғй«”дёӯж–ҮеҲҶжһҗж‘ҳиҰҒ
- **data_limitations**пјҡиіҮж–ҷйҷҗеҲ¶жҸӯйңІпјҲZero Hallucination Policyпјү

### Step 5: ж•ҙеҗҲиҲҮз¶ңеҗҲи©•дј°

Claude и®ҖеҸ– 6 д»ҪеҲҶжһҗе ұе‘ҠпјҢд»ҘеҠ ж¬Ҡж–№ејҸиЁҲз®—з¶ңеҗҲи©•еҲҶпјҡ

| ---------- | ------ | -------------------- |
| иІЎеӢҷеҲҶжһҗеё« | 25%    | еҹәжң¬йқўжҳҜй•·жңҹжҠ•иіҮж ёеҝғ |
| з”ўжҘӯзёҪз¶“   | 20%    | з”ўжҘӯи¶ЁеӢўжұәе®ҡдёӯжңҹж–№еҗ‘ |
| йҮҸеҢ–еҲҶжһҗеё« | 15%    | йўЁйҡӘйҮҸеҢ–жҸҗдҫӣе®ўи§Җеҹәжә– |
| жҠҖиЎ“еҲҶжһҗеё« | 15%    | йҖІеҮәе ҙжҷӮж©ҹеҸғиҖғ       |
| жі•дәәзұҢзўј   | 15%    | иҒ°жҳҺйҢўж–№еҗ‘жҢҮеј•       |
| ж–°иҒһжғ…з·’   | 10%    | зҹӯжңҹжғ…з·’жіўеӢ•еҸғиҖғ     |

### Step 6: з”ҹжҲҗ Dashboard

Python и…іжң¬е°Ү `integrated_report.json` иҪүеҢ–зӮәе°ҲжҘӯдә’еӢ•ејҸ HTML е„ҖиЎЁжқҝгҖӮ

### Validation Thresholds

`validate_data.py` зҸҫеңЁж”ҜжҸҙеҸҜиЁӯе®ҡй–ҫеҖјпјҢеҸҜйҖҸйҒҺ `--config config.json` иҰҶеҜ«пјҢжҲ–иҮӘеӢ•дҫқеёӮе ҙеҘ—з”Ёпјҡ

| Parameter             | Default | TW Override | Description        |
| --------------------- | ------- | ----------- | ------------------ |
| `price_freshness_days`    | 3 days  | вҖ”           | иӮЎеғ№иіҮж–ҷж–°й®®еәҰдёҠйҷҗ |
| `financial_freshness_days`| 120 days| вҖ”           | иІЎе ұиіҮж–ҷж–°й®®еәҰдёҠйҷҗ |
| `pe_min` / `pe_max`      | 0 - 500 | вҖ”           | еҗҲзҗҶ PE еҚҖй–“       |
| `single_day_change_limit` | 20%     | **11%**     | е–®ж—ҘжјІи·Ңе№…з•°еёёй–ҖжӘ» |
| `volume_spike_ratio`      | 500%    | вҖ”           | жҲҗдәӨйҮҸз•°еёёеҖҚж•ё     |
| `min_confidence_pass`     | 50      | вҖ”           | йҖҡйҒҺй©—иӯүжңҖдҪҺдҝЎеҝғеәҰ |
| `hard_stop_confidence`    | 30      | вҖ”           | дҪҺж–јжӯӨеҖјзӣҙжҺҘеҒңжӯў   |

иҮӘиЁӮй–ҫеҖјзҜ„дҫӢпјҡ

  "single_day_change_limit": 0.15,
